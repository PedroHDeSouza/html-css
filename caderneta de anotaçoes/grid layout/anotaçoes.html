grid grid container = linha por fora do grid grid lines = linhas da grid Layout grid row = linhas da grid layout grid tracks = row tracks e collumn tracks numeraçao das lines = esquerda pra direita e de cima pra baixo = valores 1 a 9 (em sentido normal) numeraçao das lines = direita pra esquerda e de baixo pra cima = valores -1 a -9 (em sentido inverso) area = ex: começa da 2 ou -6 e termina na -2 ou 6 da grid line ou 2/-2 grid gap = divisao de entre os itens (grid-gap foi depressiado, usar apenas gap) display:grid; = usado para um pai virar grid container grid-template-columns: auto auto; = ele transforma o grid em grids de duas colunas (quanto mais auto, mais colunas serao adicionadas) grid-template-rowns: 100px 100px; = cada linha cresce 100px e vao se distribuir align-items = muda os items na vertical align-content = muda os conteudos na vertical justify-items = muda os itens na horizontal justify-content = muda os conteudos na horizontal grid-auto-row = adiciona mais uma linha quando aparecer um item inesperado grid-auto-column = adiciona mais uma coluna quando aparecer um item inesperado place-items = shorthands do align-items e justify-items ex: place-items: stretch center; (primeiro sempre o align depois o justify ) place-content = shorthands do align-content e justify-content ex: place-content: start end; (primeiro o align depois o justify) grid-template = shorthands do grid-template-column e grid-template-row ex: grit-template: 100px 100px 100px / auto auto auto; (primeiro o row depois o column)
